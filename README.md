# Chat App

Simple chat application using node.js (express) and socket.io

# Run App

1. Download Repo

2. If Node is not installed - Install [Node.JS](https://nodejs.org/en/)

## Install Dependencies

```bash
npm install
```

## Run App

```bash
npm start
```
